# Adora Resume

A single-page portfolio and resume site for **Adora Okolue Mylena** (Full-Stack Content Creator). Built as plain HTML and CSS—no build step required.

## Contents

| File | Purpose |
|------|---------|
| `index.html` | Full page: layout, typography (Playfair Display + DM Sans), sections, and responsive navigation |
| `adora-favicon.svg` | Site favicon |

## Run locally

Open `index.html` in a browser, or serve the folder with any static server, for example:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

External assets: Google Fonts are loaded from `fonts.googleapis.com` (network required for typography).
